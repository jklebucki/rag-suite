# AddressBook Features

This folder contains feature slices following Vertical Slice Architecture.

Each feature owns:
- Endpoint definitions
- Request/Response models
- Handler/Service logic
- Validators (if needed)

## Features

- **CreateContact**: Create a new contact
- **GetContact**: Get a single contact by ID
- **ListContacts**: List all contacts with filtering
- **UpdateContact**: Update an existing contact
- **DeleteContact**: Delete a contact
- **SearchContacts**: Search contacts by term
- **ImportContacts**: Import contacts from CSV file (KsiazkaAdresowa.csv format)
